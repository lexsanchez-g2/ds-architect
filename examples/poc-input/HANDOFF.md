## Identity

- Rounded rectangles (not pills). `border-radius: 10px`.
- Universal 44h. WCAG-baseline compliant.
- State communicated via **border + ring**, not background.
- Disabled = bg swap + opacity-50 stacked.
- Invalid = destructive border + ring AT REST (no focus required).
- Variant axis is semantic: Password visually identical to Default; File adds 2 TEXT children.
